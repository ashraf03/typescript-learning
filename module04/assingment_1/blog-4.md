Object-Oriented Programming (OOP) helps organize and manage code in large TypeScript projects.
The four main pillars of OOP are:

Inheritance
Polymorphism
Abstraction
Encapsulation

1. Inheritance

Inheritance allows one class to reuse properties and methods from another class.

2. Polymorphism

One method can behave differently depending on the object.

3. Abstraction

Hiding complex implementation details and showing only necessary features.

4. Encapsulation

Protecting internal data and controlling access.

Usually done with:

private
protected
